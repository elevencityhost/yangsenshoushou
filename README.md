# yangsenshoushou
养森瘦瘦代理商用户管理系统
使用SpringBoot以及mybatis后台代码自动生成策略
静态资源服务器 Nginx
负载均衡 Nginx
技术选型
•	核心框架：Spring Boot 1.5.9
•	安全框架：Apache Shiro 1.3
•	视图框架：Spring MVC 4.3
•	持久层框架：MyBatis 3.3
•	定时器：Quartz 2.3
•	数据库连接池：Druid 1.0.11
•	日志管理：SLF4J 1.7、Log4j
•	页面交互：Vue2.x
•	分页插件：pagehelper 1.1.2
•	前端服务：Nginx 1.12.2
安装环境
•	JDK 1.8
•	MySql 5.1.25
•	Tomcat  8.5.23
